# Mini-Web-Browser
Simple Web Browser App  A simple web browser application built using Node.js, Express.js, HTML, and CSS. This app allows users to enter a URL, fetch the webpage content, and display it directly in the browser.



## Features

* Enter any URL in the input field
* Fetch and display webpage content
* Handles invalid URLs gracefully
* Simple and user-friendly interface

## Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* Axios
